# FE-AMANAT Starter (Vite + React)

Instruksi singkat untuk menjalankan starter project React berbasis Vite.

1. Pasang dependensi

```bash
# gunakan npm atau yarn/pnpm sesuai preferensi
npm install
```

2. Jalankan mode development

```bash
npm run dev
# buka http://localhost:5173
```

3. Build untuk produksi

```bash
npm run build
npm run preview
```

Catatan:

- Project ini minimal dan siap dikembangkan.
- Jika ingin TypeScript, saya bisa bantu konversi cepat.
